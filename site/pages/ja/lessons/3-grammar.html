言こと葉ばの形かたち (文の構造)

言こと葉ばの並ならび (語順) 述語が文末に来る。助詞が語の役割を示す。文は動詞・形容詞・名詞+だ で終わる。

私わたしは 本ほんを 読よむ

私は本を読む。

私わたしは 人ひとだ。

私は人だ。

 言いわない事こと (省略) 既知の情報は省略する。主語・目的語など、文脈から分かるものは言わなくてよい。これが日本語の基本。

「何なにを 食たべる？」「魚さかな。」

「何を食べる？」「魚。」

 ある / いる 存在がそのまま所有を表す。「have」に当たる動詞は不要。生き物には「いる」、それ以外は「ある」を使う。

猫ねこがいる。本ほんがある。

猫がいる。本がある。

私わたしは 本ほんがある。

私は本がある。

つなぐ言こと葉ば (助詞)

は (wa) 文の主題を示す。「〜については...」という意味。

この 本ほんは 良よい。/ 私わたしは 目めが 良よい。

この本は良い。/ 私は目が良い。

 が (ga) 動作の主体を示す。誰がそれをするか。新しい情報を表す。

誰だれが 来くる？

誰が来る？

 を (o) 動作の対象を示す。何に対して動作をするか。

水みずを 飲のむ。

水を飲む。

 に (ni) 方向・時間・存在の場所を表す。どこへ行くか、いつか、どこにいるか。

山やまに 行いく。/ 夜よるに 寝ねる。/ 家いえに 猫ねこがいる。

山に行く。/ 夜に寝る。/ 家に猫がいる。

 で (de) 動作の場所・手段を表す。「に」は存在の場所、「で」は動作の場所。

家いえで 読よむ。/ 手てで 書かく。

家で読む。/ 手で書く。

 の (no) 名詞と名詞をつなぐ。動詞を名詞化する用法もある（第5章参照）。

私わたしの 本ほん / 山やまの 上うえ

私の本 / 山の上

 と (to) 並列・共同・引用を表す。発言・思考・音・名前など、あらゆるものを引用できる。

水みずと 魚さかなを 買かった。/ 「行いく」と 言いった。

水と魚を買った。/ 「行く」と言った。

 も (mo) 「〜も」で同様であることを示す。は・が・を の代わりに使う。

父ちちは 本ほんを 読よむ。母ははも 読よむ。

父は本を読む。母も読む。

 から / まで 起点（から）と終点（まで）を表す。「から」は理由も表せる。

家いえから 山やままで 歩あるく。/ 夜よるまで 待まつ。

家から山まで歩く。/ 夜まで待つ。

寒さむいから、家いえにいる。

寒いから、家にいる。

 より 「Xより」で「Xよりも」の意味。この助詞一つであらゆる比較を表せる。

山やまより 大おおきい。/ 魚さかなより 肉にくが 好すきだ。

山より大きい。/ 魚より肉が好きだ。

する事ことの形かたち (動詞の活用)

四よっつの形かたち 普通形のみ扱う。丁寧形（ます）は社会的な層であり文法ではない。る/た/ない/なかった。すべてこの上に構築する。

食たべる → 食たべた → 食たべない → 食たべなかった

食べる → 食べた → 食べない → 食べなかった

行いく → 行いった → 行いかない → 行いかなかった

行く → 行った → 行かない → 行かなかった

する → した → しない → しなかった

する → した → しない → しなかった

来くる → 来きた → 来こない → 来こなかった

来る → 来た → 来ない → 来なかった

 て形けい 万能の接続形。動作の連結、依頼（+ください）、進行（+いる）、許可（+もいい）、禁止（+はいけない）。時制は文末のみ。活用: く→いて、む→んで、つ→って、る→て。

書かいて 読よんで 寝ねた。

書いて読んで寝た。

待まってください。/ 今いま 読よんでいる。

待ってください。/ 今読んでいる。

座すわってもいい？ / 入はいってはいけない。

座ってもいい？ / 入ってはいけない。

書かく→書かいて / 飲のむ→飲のんで / 待まつ→待まって / 食たべる→食たべて

書く→書いて / 飲む→飲んで / 待つ→待って / 食べる→食べて

 られる / える 可能を表す。一段動詞: る を取って られる を付ける。五段動詞: え段 + る。する → できる。

書かける / 見みられる / する → できる

書ける / 見られる / する → できる

どんな (修飾)

い のどんな 動詞と同様に活用する。「寒い」だけで文が成立する。活用: くない・かった・くて。「だ」は不要。

寒さむい。寒さむくない。寒さむかった。寒さむくて 暗くらい。

寒い。寒くない。寒かった。寒くて暗い。

 な のどんな 名詞の前に「な」を付ける。否定は「じゃない」、過去は「だった」。

必要ひつような 物もの / 必要ひつようじゃない / 必要ひつようだった

必要な物 / 必要じゃない / 必要だった

 〜くなる / 〜になる 状態の変化を表す。あらゆる変化をこの一つの型で表現できる。

暑あつくなる。/ 必要ひつようになる。

暑くなる。/ 必要になる。

 する事こと + 物もの 動詞や文を名詞の前に置いて修飾する。語彙を作り出す力の源。

山やまで 遊あそぶ 子供こども / 私わたしが 買かった 物もの

山で遊ぶ子供 / 私が買った物

物ものにする (名詞化)

の くだけた名詞化。「食べるのが好き」。疑問文にも使う:「何しているの？」

食たべるのが 好すきだ。/ 何なにしているの？

食べるのが好きだ。/ 何しているの？

 こと 動作を抽象的な概念にする。「走ること」は走るという考え。「飲むもの」は物理的な物。「こと」は抽象、「もの」は具体。

走はしることは良よい。

走ることは良い。

行いくことができる。

行くことができる。

大切たいせつなことを言いいます。

大切なことを言います。

 もの 触れたり指せたりする物理的なもの。「飲むもの」= 飲める何か。「こと」は抽象的な概念、「もの」は具体的な物。

飲のむもの / 読よむもの

飲むもの / 読むもの

大切たいせつなものを持もっています。

大切なものを持っています。

 方かた やり方を表す。動詞の連用形に付ける。食べ方 = 食べる方法。使い方 = 使う方法。こと（概念）+ もの（物）+ 方（方法）で何でも表現できる。

食たべ方かたを教おしえてください。

食べ方を教えてください。

この言こと葉ばの使つかい方かたが分わかりません。

この言葉の使い方が分かりません。

聞きき方かたが大切たいせつです。

聞き方が大切です。

思おもうと「もし」 (意志と条件)

たい / 欲ほしい 動作の希望（動詞の連用形 + たい）。物の希望（欲しい）。

行いきたい。/ 新あたらしい 家いえが 欲ほしい。

行きたい。/ 新しい家が欲しい。

 たら / なら たら = もしその事態が起こったら（動詞のた形 + ら）。なら = もしそうであるなら。

雨あめが 降ふったら、行いかない。/ 時間じかんがあるなら、来きてください。

雨が降ったら、行かない。/ 時間があるなら、来てください。

 と思おもう / と言いう 「と」で何でも引用できる。思考・発言・音。「という」は「〜と呼ばれる」の意味。

良よいと 思おもう。/ 「行いく」と 言いった。/ 富士山ふじさんという 山やま

良いと思う。/ 「行く」と言った。/ 富士山という山

 ために 目的と受益者を表す。動詞+ために、または名詞+のために。

学まなぶために 読よむ。/ 子供こどものために。

学ぶために読む。/ 子供のために。

聞きき方かた (疑問と終助詞)

か / ね / よ 疑問（か）/ 同意の確認（ね）/ 主張（よ）。くだけた会話では「か」を省略して語尾を上げるだけのことも多い。

これは 水みずか？ / 良よい 日ひだね。/ 行いくよ。

これは水か？ / 良い日だね。/ 行くよ。

 何なに・誰だれ・どこ・いつ・なぜ + も/でも 疑問詞は答えの位置に置く。+も（否定）= 何もない・誰もいない。+でも = 何でも・誰でも。疑問詞5つ + 助詞2つ = 20語の代名詞が作れる。

何なにを 食たべる？→ 何なにも 食たべない。

何を食べる？→ 何も食べない。

何なにでも 作つくれる。/ 誰だれもいない。/ 誰だれでもできる。

何でも作れる。/ 誰もいない。/ 誰でもできる。
