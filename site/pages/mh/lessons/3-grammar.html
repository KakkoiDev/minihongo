言こと葉ばの形かたち

言こと葉ばの並ならび 言こと葉ばの最さい後ごにする事ことが来くる。つなぐ言こと葉ばが何なにをするか見みせる。する事ことか、どんなか、物もの+だ で終おわる。

私わたしは 本ほんを 読よむ

私わたしは 人ひとだ。

 言いわない事こと 知しっている事ことは言いわない。誰だれが、何なにを - 分わかるなら言いわなくていい。これが普通ふつう。

「何なにを 食たべる？」「魚さかな。」

 ある / いる ある事ことは持もつ事ことと同おなじ。「持もつ」のする事ことはない。生いき物ものはいる、他ほかは全部ぜんぶある。

猫ねこがいる。本ほんがある。

私わたしは 本ほんがある。

つなぐ言こと葉ば

は 何なにの話はなしかを見みせる。「これの事ことは...」

この 本ほんは 良よい。/ 私わたしは 目めが 良よい。

 が する人ひと。誰だれがそれをするか。新あたらしい事こと。

誰だれが 来くる？

 を 何なににする事ことをするか。

水みずを 飲のむ。

 に どこに行いく。いつ。どこにいる。

山やまに 行いく。/ 夜よるに 寝ねる。/ 家いえに 猫ねこがいる。

 で どこでする。何なにを使つかってする。に は物ものがいるところ、で はする事ことのところ。

家いえで 読よむ。/ 手てで 書かく。

 の 物ものと物ものをつなぐ。する事ことを物ものにもする (第だい5を見みる)。

私わたしの 本ほん / 山やまの 上うえ

 と 「と」でつなぐ。「と一いっ緒しょに」。言いった事ことを書かく。言いう事こと、思おもう事こと、音おと、名前なまえ。

水みずと 魚さかなを 買かった。/ 「行いく」と 言いった。

 も 「も」で同おなじ。は、が、を の代かわりに使つかう。

父ちちは 本ほんを 読よむ。母ははも 読よむ。

 から / まで 始はじまるところ (から)、終おわるところ (まで)。から はなぜかも言いえる。

家いえから 山やままで 歩あるく。/ 夜よるまで 待まつ。

寒さむいから、家いえにいる。

 より Xより = 「Xよりもっと」。この一ひとつの言こと葉ばで全部ぜんぶ。

山やまより 大おおきい。/ 魚さかなより 肉にくが 好すきだ。

する事ことの形かたち

四よっつの形かたち 四よっつの形かたちがある。る = 今いまと後あと。た = 前まえ。ない = しない事こと。なかった = 前まえにしなかった事こと。ます は人ひとへの言いい方かたで、言葉ことばの道みちではない。全部ぜんぶこの四よっつの上うえに作つくる。

食たべる → 食たべた → 食たべない → 食たべなかった

行いく → 行いった → 行いかない → 行いかなかった

する → した → しない → しなかった

来くる → 来きた → 来こない → 来こなかった

 て形けい 全部ぜんぶをつなぐ形かたち。する事ことをつなぐ、お願ねがい (+ください)、今いましている (+いる)、していい (+もいい)、してはいけない (+はいけない)。時ときは最さい後ごだけ。く→いて、む→んで、つ→って、る→て。

書かいて 読よんで 寝ねた。

待まってください。/ 今いま 読よんでいる。

座すわってもいい？ / 入はいってはいけない。

書かく→書かいて / 飲のむ→飲のんで / 待まつ→待まって / 食たべる→食たべて

 られる / える できる事こと。る-のする事こと: る を取とって、られる を足たす。他ほか: え の並ならび + る。する → できる。

書かける / 見みられる / する → できる

どんな

い のどんな する事ことと同おなじ。寒さむい だけで言こと葉ばが終おわる。変かえる: くない、かった、くて。だ はいらない。

寒さむい。寒さむくない。寒さむかった。寒さむくて 暗くらい。

 な のどんな 物ものの前まえに な を足たす。じゃない (違ちがう)、だった (前まえ)。

必要ひつような 物もの / 必要ひつようじゃない / 必要ひつようだった

 〜くなる / 〜になる 変かわる事こと。全部ぜんぶの変かわり方かたがこれ一ひとつ。

暑あつくなる。/ 必要ひつようになる。

 する事こと + 物もの する事ことや言こと葉ばを物ものの前まえに置おいて、それを見みせる。言こと葉ば作づくりの力ちから。

山やまで 遊あそぶ 子供こども / 私わたしが 買かった 物もの

物ものにする

の 普通ふつうの形かたち。食たべるのが好すき。聞きく時ときにも: 何なにしているの？

食たべるのが 好すきだ。/ 何なにしているの？

 こと する事ことを考かんがえにする。目めで見みえない事こと。「走はしる事こと」は考かんがえ。「飲のむ物もの」は手てで持もてる。この違ちがいが大切たいせつ。

走はしることは良よい。

行いくことができる。

大切たいせつなことを言いいます。

 もの 手てで持もてるもの。本当ほんとうにある何なにか。「飲のむもの」= 飲のめる何なにか。「こと」は考かんがえ、「もの」は物もの。

飲のむもの / 読よむもの

大切たいせつなものを持もっています。

 方かた どうするかの形かたち。する事ことの始はじめに付つける。食たべ方かた= どう食たべるか。使つかい方かた= どう使つかうか。こと・もの・方かたの三みっつで全部ぜんぶ言いえる。

食たべ方かたを教おしえてください。

この言こと葉ばの使つかい方かたが分わかりません。

聞きき方かたが大切たいせつです。

思おもうと「もし」

たい / 欲ほしい する事ことがしたい (する事ことの始はじめ + たい)。物ものが欲ほしい (欲ほしい)。

行いきたい。/ 新あたらしい 家いえが 欲ほしい。

 たら / なら たら = もしそれが起おこったら (した + ら)。なら = もしそうなら。

雨あめが 降ふったら、行いかない。/ 時間じかんがあるなら、来きてください。

 と思おもう / と言いう と で何なにでも言いえる。思おもう事こと、言いう事こと、音おと。という = 「その名前なまえの」。

良よいと 思おもう。/ 「行いく」と 言いった。/ 富士山ふじさんという 山やま

 ために 何なにのためにするか。する事こと+ために か 物もの+のために。

学まなぶために 読よむ。/ 子供こどものために。

聞きき方かた

か / ね / よ 聞きく / そうでしょう？ / 強つよく言いう。普通ふつうの話はなしでは か を言いわない事ことも多おおい。

これは 水みずか？ / 良よい 日ひだね。/ 行いくよ。

 何なに・誰だれ・どこ・いつ・なぜ + も/でも 聞きく言こと葉ばは答こたえのところに置おく。+も (ない) = 何なにもない。+でも = 何なにでも。5つの言こと葉ば + 2つのつなぎ = 20の言こと葉ばが作つくれる。

何なにを 食たべる？→ 何なにも 食たべない。

何なにでも 作つくれる。/ 誰だれもいない。/ 誰だれでもできる。
