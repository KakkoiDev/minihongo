第1: 文の形

語順 (Word Order) Verb last. Particles mark roles. Sentences end with verb, adjective, or noun+だ.

私わたしは 本ほんを 読よむ

私【わたし】は 本【ほん】を 読【よ】む

私わたしは 人ひとだ。

私【わたし】は 人【ひと】だ。

 省略 (Dropping) Drop what's known. Subjects, objects - anything established vanishes. This is the default.

「何なにを 食たべる？」「魚さかな。」

「何【なに】を 食【た】べる？」「魚【さかな】。」

 ある / いる (Existence) Existence IS possession. No separate "have" verb. いる for living, ある for everything else.

猫ねこがいる。本ほんがある。

猫【ねこ】がいる。本【ほん】がある。

私わたしは 本ほんがある。

私【わたし】は 本【ほん】がある。

第2: 助詞

は (wa) — topic Topic marker. "As for X..."

この 本ほんは 良よい。/ 私わたしは 目めが 良よい。

この 本【ほん】は 良【よ】い。/ 私【わたし】は 目【め】が 良【よ】い。

 が (ga) — subject Subject. Who does it. New information.

誰だれが 来くる？

誰【だれ】が 来【く】る？

 を (o) — object Object. What the action is done to.

水みずを 飲のむ。

水【みず】を 飲【の】む。

 に (ni) — direction / time / location Direction, time, location of existence.

山やまに 行いく。/ 夜よるに 寝ねる。/ 家いえに 猫ねこがいる。

山【やま】に 行【い】く。/ 夜【よる】に 寝【ね】る。/ 家【いえ】に 猫【ねこ】がいる。

 で (de) — location of action / means Location of action, means/tool. Compare に: に = where things exist, で = where things happen.

家いえで 読よむ。/ 手てで 書かく。

家【いえ】で 読【よ】む。/ 手【て】で 書【か】く。

 の (no) — connecting Connects nouns. Also turns verbs into nouns (see section 5).

私わたしの 本ほん / 山やまの 上うえ

私【わたし】の 本【ほん】 / 山【やま】の 上【うえ】

 と (to) — and / quoting And, with, quotation. Quotes anything: speech, thoughts, sounds, names.

水みずと 魚さかなを 買かった。/ 「行いく」と 言いった。

水【みず】と 魚【さかな】を 買【か】った。/ 「行【い】く」と 言【い】った。

 も (mo) — also Also. Replaces は, が, or を.

父ちちは 本ほんを 読よむ。母ははも 読よむ。

父【ちち】は 本【ほん】を 読【よ】む。母【はは】も 読【よ】む。

 から / まで — from / until Start points (from), end points (until). から also gives reasons (because).

家いえから 山やままで 歩あるく。/ 夜よるまで 待まつ。

家【いえ】から 山【やま】まで 歩【ある】く。/ 夜【よる】まで 待【ま】つ。

寒さむいから、家いえにいる。

寒【さむ】いから、家【いえ】にいる。

 より — comparison Xより = "more than X." One particle, all comparisons.

山やまより 大おおきい。/ 魚さかなより 肉にくが 好すきだ。

山【やま】より 大【おお】きい。/ 魚【さかな】より 肉【にく】が 好【す】きだ。

第3: 動詞

四つの形 (Four Forms) Plain form only. Polite form (ます) is a social layer, not grammar. る/た/ない/なかった. Everything layers on top.

食たべる → 食たべた → 食たべない → 食たべなかった

食【た】べる → 食【た】べた → 食【た】べない → 食【た】べなかった

行いく → 行いった → 行いかない → 行いかなかった

行【い】く → 行【い】った → 行【い】かない → 行【い】かなかった

する → した → しない → しなかった

する → した → しない → しなかった

来くる → 来きた → 来こない → 来こなかった

来【く】る → 来【き】た → 来【こ】ない → 来【こ】なかった

 て形 (Te-Form) Universal connector. Chain actions, request (+ください), ongoing (+いる), permission (+もいい), prohibition (+はいけない). One tense at the end. Formation varies by verb ending: く→いて, む→んで, つ→って, る→て.

書かいて 読よんで 寝ねた。

書【か】いて 読【よ】んで 寝【ね】た。

待まってください。/ 今いま 読よんでいる。

待【ま】ってください。/ 今【いま】 読【よ】んでいる。

座すわってもいい？ / 入はいってはいけない。

座【すわ】ってもいい？ / 入【はい】ってはいけない。

書かく→書かいて / 飲のむ→飲のんで / 待まつ→待まって / 食たべる→食たべて

書【か】く→書【か】いて / 飲【の】む→飲【の】んで / 待【ま】つ→待【ま】って / 食【た】べる→食【た】べて

 られる / える (Potential) Ability. る-verbs: drop る, add られる. Other verbs: え-row + る. する → できる.

書かける / 見みられる / する → できる

書【か】ける / 見【み】られる / する → できる

第4: 様子

い形容詞 ARE verbs. 寒い is a complete sentence. Conjugate: くない, かった, くて. No copula needed.

寒さむい。寒さむくない。寒さむかった。寒さむくて 暗くらい。

寒【さむ】い。寒【さむ】くない。寒【さむ】かった。寒【さむ】くて 暗【くら】い。

 な形容詞 Add な before nouns. じゃない (neg), だった (past).

必要ひつような 物もの / 必要ひつようじゃない / 必要ひつようだった

必要【ひつよう】な 物【もの】 / 必要【ひつよう】じゃない / 必要【ひつよう】だった

 〜くなる / 〜になる Change of state. One pattern for all changes.

暑あつくなる。/ 必要ひつようになる。

暑【あつ】くなる。/ 必要【ひつよう】になる。

 Clause + noun (Relative Clauses) Put a verb or sentence before a noun to describe it. Engine of word-building.

山やまで 遊あそぶ 子供こども / 私わたしが 買かった 物もの

山【やま】で 遊【あそ】ぶ 子供【こども】 / 私【わたし】が 買【か】った 物【もの】

第5: 名詞化

の (Nominalization) Casual. 食べるのが好き. Also in questions: 何しているの？

食たべるのが 好すきだ。/ 何なにしているの？

食【た】べるのが 好【す】きだ。/ 何【なに】しているの？

 こと Abstract concept. 走ることは良い. 行くことができる.

走はしることは 良よい。/ 行いくことができる。

走【はし】ることは 良【よ】い。/ 行【い】くことができる。

 もの Physical thing or tangible reason. 飲むもの = something to drink.

飲のむもの / 読よむもの

飲【の】むもの / 読【よ】むもの

第6: 思いと条件

たい / 欲しい Want to do (verb stem + たい). Want a thing (欲しい).

行いきたい。/ 新あたらしい 家いえが 欲ほしい。

行【い】きたい。/ 新【あたら】しい 家【いえ】が 欲【ほ】しい。

 たら / なら たら = when/if something happens (verb た + ら). なら = if it's the case.

雨あめが 降ふったら、行いかない。/ 時間じかんがあるなら、来きてください。

雨【あめ】が 降【ふ】ったら、行【い】かない。/ 時間【じかん】があるなら、来【き】てください。

 と思おもう / と言いう Quote anything with と. Thoughts, speech, sounds. という = "called."

良よいと 思おもう。/ 「行いく」と 言いった。/ 富士山ふじさんという 山やま

良【よ】いと 思【おも】う。/ 「行【い】く」と 言【い】った。/ 富士山【ふじさん】という 山【やま】

 ために Purpose and beneficiary. Verb+ために or noun+のために.

学まなぶために 読よむ。/ 子供こどものために。

学【まな】ぶために 読【よ】む。/ 子供【こども】のために。

第7: 聞き方

か / ね / よ Question / right? / emphasis. か often dropped in casual - just rising tone.

これは 水みずか？ / 良よい 日ひだね。/ 行いくよ。

これは 水【みず】か？ / 良【よ】い 日【ひ】だね。/ 行【い】くよ。

 何・誰・どこ・いつ・なぜ + も/でも Question words go where the answer goes. +も (neg verb) = nothing/nobody. +でも = anything/anyone. 5 words + 2 particles = 20 pronouns free.

何なにを 食たべる？→ 何なにも 食たべない。

何【なに】を 食【た】べる？→ 何【なに】も 食【た】べない。

何なにでも 作つくれる。/ 誰だれもいない。/ 誰だれでもできる。

何【なに】でも 作【つく】れる。/ 誰【だれ】もいない。/ 誰【だれ】でもできる。
