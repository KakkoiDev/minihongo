Grammar — Minihongo

Grammar

Minimum grammar for maximum expression.

第1: 文の作り方 (Sentence Basics)

語順 (Word Order) The verb always comes last. Subject - Object - Verb.

私わたしは 本ほんを 読よみます

I read a book. (I + book + read)

 です / だ Equates two things. です = polite, だ = casual. じゃない = negative. でした = past.

これは本ほんです → 本ほんじゃない → 本ほんでした

This is a book → not a book → was a book

第2: 助詞 (Particles)

は (wa) — topic Marks what you are talking about.

この本ほんは良よいです。

This book is good.

 が (ga) — subject Marks who/what does the action.

誰だれが来きますか？

Who is coming?

 を (o) — object Marks what the action is done to.

水みずを飲のみます。

I drink water.

 に (ni) — direction / time / location Where you go, when something happens, where something exists.

山やまに行いきます。 / 夜よるに食たべます。 / 家いえに人ひとがいます。

I go to the mountain. / I eat at night. / There is a person in the house.

 で (de) — location of action / means Where you do something, or what you use to do it.

家いえで読よみます。 / 手てで食たべます。

I read at home. / I eat with my hands.

 の (no) — possession Connects nouns. Like 's in English.

私わたしの本ほん

My book

 も (mo) — also Replaces は, が, or を to mean "also."

父ちちは本ほんを読よみます。母ははも読よみます。

Father reads. Mother also reads.

 と (to) — and Lists items together.

水みずと本ほんを買かいました。

I bought water and a book.

 から (kara) — from / because Starting point (place/time), or reason.

家いえから行いきます。 / 寒さむいから、家いえにいます。

I go from home. / Because it's cold, I stay home.

 まで (made) — until / to End point in time or place. Pairs with から.

家いえから山やままで行いきます。 / 夜よるまで待まちます。

I go from home to the mountain. / I wait until night.

第3: 動詞の形 (Verb Forms)

三つの種類 (Three Types) U-verbs: end in う-row. Ru-verbs: end in える/いる. Irregular: する, 来る.

う: 買かう、書かく、探さがす — る: 食たべる、見みる

U-verbs: buy, write, search — Ru-verbs: eat, see

 ます形 (Polite) ます / ません / ました. ましょう = let's. Ru: drop る, add ます. U: い-row + ます.

食たべます / 食たべません / 食たべました / 食たべましょう

eat / don't eat / ate / let's eat

 て形 (Te-Form) Connects actions, makes requests (て+ください), shows ongoing state (て+いる). Change て→た for past tense.

る → て: 食たべて / う・つ・る → って: 買かって / む・ぶ → んで: 飲のんで / く → いて: 書かいて

eat → / buy → / drink → / write →

待まってください。 / 今いま、読よんでいます。 / 食たべた。

Please wait. / I am reading now. / I ate. (て→た = past)

 ない形 (Negative) Ru: drop る, add ない. U: あ-row + ない. ある → ない.

食たべない / 買かわない / 書かかない

don't eat / don't buy / don't write

 られる形 (Potential) Ru: drop る, add られる. U: え-row + る. する → できる. Expresses ability.

食たべられる / 行いける / 見みえる

can eat / can go / can see

第4: 様子の言葉 (Adjectives)

い形容詞 End in い. Change い → くない (neg), かった (past), くて (and).

大おおきい → 大おおきくない → 大おおきかった

big → not big → was big

 な形容詞 Add な before nouns. じゃない (neg), だった (past).

必要ひつような物もの / 必要ひつようじゃない / 必要ひつようだった

necessary thing / not necessary / was necessary

 〜くなる / 〜になる い-adj: い → くなる. な-adj/noun: + になる. Change of state.

暑あつくなります。 / 必要ひつようになります。

It becomes hot. / It becomes necessary.

 〜すぎる Verb stem + すぎる. い-adj: drop い + すぎる. Too much of something.

食たべすぎます。 / 高たかすぎます。

I eat too much. / It is too expensive.

第5: 使い方 (Patterns)

〜たい / 欲しい Verb stem + たい = want to do. 欲しい = want a thing.

行いきたい / 新あたらしい本ほんが欲ほしい

I want to go / I want a new book

 たら / なら たら = when/if something happens (verb た + ら). なら = if it's the case.

雨あめが降ふったら、行いきません。 / 時ときがあるなら、来きてください。

If it rains, I won't go. / If you have time, please come.

 と思おもう / と言いう sentence + と + 思う (think) or 言う (say). Reports thoughts and speech.

良よいと思おもいます。 / 「行いく」と言いいました。

I think it is good. / They said "I'll go."

 てもいい / てはいけない て + も + いい = may do. て + は + いけない = must not do.

座すわってもいいですか？ / 入はいってはいけません。

May I sit? / You must not enter.

第6: 聞き方 (Questions)

か / ね / よ か = question. ね = right? よ = you know!

これは本ほんですか？ / 良よい日ひですね。

Is this a book? / Nice day, isn't it?

 何・誰・どこ・いつ・なぜ Place the question word where the answer would go.

何なにを食たべますか？ / 山やまはどこですか？

What will you eat? / Where is the mountain?

第7: 修飾 (Modification)

verb + noun (Relative Clauses) Put a verb or sentence before a noun to describe it. The core of word-building.

教おしえる人ひと = 先生せんせい / 買かう場所ばしょ = 店みせ

person who teaches = teacher / place where you buy = shop

本ほんを読よむ人ひと

person who reads books
