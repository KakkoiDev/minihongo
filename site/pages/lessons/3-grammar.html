文法 — Minihongo

文法 (Grammar)

Essential Japanese grammar patterns, explained simply.

第1: 文の作り方 (Sentence Basics)

語順 (Word Order) The verb always comes last. Subject - Object - Verb.

私わたしは 本ほんを 読よみます

I read a book. (I + book + read)

 です (polite) / だ (casual) Equates two things. です is polite, だ is casual.

私わたしは先生せんせいです。→ 先生せんせいじゃありません → 先生せんせいでした

I am a teacher. → Not a teacher. → Was a teacher.

第2: 小さい言葉 (Particles)

は (wa) Topic marker — marks what you are talking about.

私わたしは先生せんせいです。

I am a teacher. (As for me, teacher.)

 が (ga) Subject marker — marks who/what does the action.

誰だれが来きますか？

Who is coming?

 を (o) Object marker — marks what the action is done to.

水みずを飲のみます。

I drink water.

 に (ni) Direction, time, or location (with いる/ある).

学校がっこうに行いきます。

I go to school.

朝あさに食たべます。

I eat in the morning.

家いえに人ひとがいます。

There is a person in the house.

 で (de) Location of action, or means/tool used.

家いえで学まなびます。

I study at home.

電話でんわで話はなします。

I talk on the phone.

 の (no) Possession or connection — like 's in English.

私わたしの本ほん

My book

 も (mo) Also, too — replaces は, が, or を.

友達ともだちは先生せんせいです。私わたしも先生せんせいです。

My friend is a teacher. I am also a teacher.

 と (to) And (listing items).

水みずと本ほんを買かいました。

I bought water and a book.

 から (kara) From (place/time), or because (reason).

家いえから学校がっこうまで歩あるきます。

I walk from home to school.

寒さむいから、家いえにいます。

Because it's cold, I stay home.

 まで (made) Until (time or place).

今日きょうまで待まちます。

I will wait until today.

 か (ka) / ね (ne) / よ (yo) か = question. ね = seeking agreement ("right?"). よ = asserting ("you know!").

先生せんせいですか？

Are you a teacher?

良よい日ひですね。

Nice day, isn't it?

第3: 動詞の形 (Verb Forms)

三つの種類 (Three Verb Types)

う言葉 / る言葉 / 変な言葉 U-verbs end in う/く/す/つ/ぬ/ぶ/む/る. Ru-verbs end in える/いる. Irregular: する and 来る only.

う: 買かう、書かく、話はなす

U-verbs: buy, write, speak

る: 食たべる、見みる

Ru-verbs: eat, see

ます形 (Polite Form)

ます / ません / ました / ませんでした Polite present/negative/past/past-negative. Ru-verbs: drop る, add ます. U-verbs: change last sound to い-row, add ます.

食たべる → 食たべます / 食たべません / 食たべました

eat → eat (polite) / don't eat / ate

買かう → 買かいます / 書かく → 書かきます / する → します

buy → buy (polite) / write → write (polite) / do → do (polite)

て形 (Te-Form)

て形の作り方 Connects actions, makes requests, shows ongoing state. Ru-verbs: drop る, add て. U-verbs: sound changes apply.

る: 食たべる → 食たべて / 見みる → 見みて

Ru-verbs: eat → eating / see → seeing

う/つ/る → って: 買かう → 買かって

U-verbs ending う/つ/る → って

む/ぶ/ぬ → んで: 飲のむ → 飲のんで

U-verbs ending む/ぶ/ぬ → んで

く → いて: 書かく → 書かいて / す → して: 話はなす → 話はなして

write → writing / speak → speaking

 て + ください Please do ~.

少すこし待まってください。

Please wait a moment.

 て + いる Ongoing action or resulting state.

今いま、学まなんでいます。

I am studying now.

 て + もいい / はいけない もいい = may I? はいけない = must not.

食たべてもいいですか？

May I eat?

ここで走はしってはいけません。

You must not run here.

た形 / ない形 (Past / Negative)

た形 Past tense. Same sound changes as て-form, but て→た, で→だ.

食たべて → 食たべた / 飲のんで → 飲のんだ / 書かいて → 書かいた

ate / drank / wrote

 ない形 Negative. Ru-verbs: drop る, add ない. U-verbs: change last sound to あ-row, add ない. Exception: ある → ない.

食たべる → 食たべない / 買かう → 買かわない / 書かく → 書かかない

don't eat / don't buy / don't write

 できる形 (Potential) Can do ~. Ru-verbs: drop る, add られる. U-verbs: change to え-row, add る. する → できる.

言葉ことばが話はなせます。

I can speak the language.

第4: 様子の言葉 (Adjectives)

い形容詞 (i-Adjectives) End in い. Conjugate by changing い: くない (neg), かった (past), くて (and).

大おおきい → 大おおきくない → 大おおきかった → 大おおきくて

big → not big → was big → big and...

 な形容詞 (na-Adjectives) Add な before nouns. Use じゃない (neg), だった (past), で (and).

大切たいせつな物もの

An important thing

大切たいせつ → 大切たいせつじゃない → 大切たいせつだった → 大切たいせつで

important → not important → was important → important and...

第5: 大切な使い方 (Key Patterns)

〜たい / 欲しい (Wanting) Verb stem + たい = want to do. 欲しい = want a thing.

国くにに行いきたいです。

I want to go to the country.

新あたらしいコンピューターが欲ほしいです。

I want a new computer.

 〜なければならない (Must) ない-form stem + なければならない. Short form: なきゃ.

行いかなければなりません。

I must go.

 与える / もらう (Giving & Receiving) 与える = give. もらう = receive.

友達ともだちに本ほんを与あたえました。

I gave a book to my friend.

友達ともだちから本ほんをもらいました。

I received a book from my friend.

 もっと / 同じ (Comparison) もっと = more. 同じ = same.

これはもっと大おおきいです。

This is bigger.

これとそれは同おなじです。

This and that are the same.

 〜ましょう (Let's ~) Polite suggestion to do something together.

食たべましょう。

Let's eat.

 なら (If) Conditional — if something is true.

時ときがあるなら、来きてください。

If you have time, please come.

第6: 聞く (Questions)

はい / か Questions Add か to the end of a statement.

先生せんせいですか？ — はい、先生せんせいです。

Are you a teacher? — Yes, I am a teacher.

 何・誰・どこ・いつ・なぜ (WH-Questions) Place the question word where the answer would go.

何なにを食たべますか？

What will you eat?

学校がっこうはどこですか？

Where is the school?
