文法 — Minihongo

文法 (Grammar)

Essential Japanese grammar patterns, explained simply.

第1: 文を作る (Building Sentences)

語順 (Word Order) The verb always comes last. Subject - Object - Verb.

私わたしは 食たべ物ものを 食たべます

I eat food. (I + food + eat)

第2: 小さい言葉 (Particles)

は (wa) Topic marker — marks what you are talking about.

私わたしは学生がくせいです。

I am a student. (As for me, student.)

 が (ga) Subject marker — marks who/what does the action.

誰だれが来きますか？

Who is coming?

 を (o) Object marker — marks what the action is done to.

食たべ物ものを食たべます。

I eat food.

 に (ni) Direction, time, or location (with いる/ある).

学校がっこうに行いきます。

I go to school.

朝あさに食たべます。

I eat in the morning.

家いえに人ひとがいます。

There is a person in the house.

 で (de) Location of action, or means/tool used.

家いえで学まなびます。

I study at home.

電話でんわで話はなします。

I talk on the phone.

 の (no) Possession or connection — like 's in English.

私わたしの本ほん

My book

国くにの食たべ物もの

The country's food

 も (mo) Also, too — replaces は, が, or を.

友達ともだちは学生がくせいです。私わたしも学生がくせいです。

My friend is a student. I am also a student.

 と (to) And (exhaustive list — only these items).

水みずと食たべ物ものを買かいました。

I bought water and food. (Only those two.)

 や (ya) And (partial list — these and more).

水みずや食たべ物ものを買かいました。

I bought water, food, etc.

 から (kara) From (place/time), or because (reason).

家いえから学校がっこうまで歩あるきます。

I walk from home to school.

仕事しごとがあるから、行いけません。

I can't go because I have work.

 まで (made) Until (time or place).

今日きょうまで待まちます。

I will wait until today.

 か (ka) Question marker — placed at end of sentence.

学生がくせいですか？

Are you a student?

 ね (ne) / よ (yo) ね = seeking agreement ("right?"). よ = asserting information ("you know!").

良よい日ひですね。

Nice day, isn't it?

今いまは朝あさですよ。

It's morning, you know!

 しか (shika) Only — always used with negative verb.

お金かねが少すこししかありません。

I only have a little money.

第3: 動詞の形 (Verb Forms)

三つの種類 (Three Verb Types)

う言葉 / る言葉 / 変な言葉 U-verbs end in う/く/す/つ/ぬ/ぶ/む/る. Ru-verbs end in える/いる. Irregular: する and 来る only.

う: 買かう、書かく、話はなす

U-verbs: buy, write, speak

る: 食たべる、見みる

Ru-verbs: eat, see

ます形 (Polite Form)

ます / ません / ました / ませんでした Polite present/negative/past/past-negative. Ru-verbs: drop る, add ます. U-verbs: change last sound to い-row, add ます.

食たべる → 食たべます / 食たべません / 食たべました / 食たべませんでした

eat → eat (polite) / don't eat / ate / didn't eat

買かう → 買かいます / 書かく → 書かきます / する → します

buy → buy (polite) / write → write (polite) / do → do (polite)

て形 (Te-Form)

て形の作り方 Connects actions, makes requests, shows ongoing state. Ru-verbs: drop る, add て. U-verbs: sound changes apply.

る: 食たべる → 食たべて / 見みる → 見みて

Ru-verbs: eat → eating / see → seeing

う/つ/る → って: 買かう → 買かって

U-verbs ending う/つ/る → って: buy → buying

む/ぶ/ぬ → んで: 飲のむ → 飲のんで

U-verbs ending む/ぶ/ぬ → んで: drink → drinking

く → いて: 書かく → 書かいて / す → して: 話はなす → 話はなして

write → writing / speak → speaking

て形の使い方 (Te-Form Uses)

て + ください Please do ~. Polite request.

少すこし待まってください。

Please wait a moment.

 て + いる Ongoing action or resulting state.

今いま、学まなんでいます。

I am studying now.

 て + もいいですか May I ~? Asking permission.

食たべてもいいですか？

May I eat?

 て + はいけない Must not ~. Prohibition.

ここで写真しゃしんを取とってはいけません。

You must not take photos here.

 て + みる Try doing ~.

この食たべ物ものを食たべてみます。

I'll try eating this food.

た形 / ない形 (Past / Negative)

た形 Past tense. Same sound changes as て-form, but て→た, で→だ.

食たべて → 食たべた / 飲のんで → 飲のんだ / 書かいて → 書かいた

ate / drank / wrote

 ない形 Negative. Ru-verbs: drop る, add ない. U-verbs: change last sound to あ-row, add ない. Exception: ある → ない.

食たべる → 食たべない / 買かう → 買かわない / 書かく → 書かかない

don't eat / don't buy / don't write

 できる形 (Potential) Can do ~. Ru-verbs: drop る, add られる. U-verbs: change to え-row, add る. する → できる.

言語げんごが話はなせます。

I can speak the language.

第4: 様子の言葉 (Adjectives)

い形容詞 (i-Adjectives) End in い. Conjugate by changing い: くない (neg), かった (past), くて (and).

大おおきい → 大おおきくない → 大おおきかった → 大おおきくて

big → not big → was big → big and...

 な形容詞 (na-Adjectives) Add な before nouns. Use じゃない (neg), だった (past), で (and).

便利べんりな コンピューター

A convenient computer

便利べんり → 便利べんりじゃない → 便利べんりだった → 便利べんりで

convenient → not convenient → was convenient → convenient and...

第5: です と だ (Copula)

です (polite) / だ (casual) Equates nouns. です is polite, だ is casual.

私わたしは学生がくせいです。→ 学生がくせいじゃありません → 学生がくせいでした

I am a student. → Not a student. → Was a student.

学生がくせいだ。→ 学生がくせいじゃない。→ 学生がくせいだった。

Am a student. → Not a student. → Was a student. (casual)

第6: 大切な使い方 (Important Patterns)

〜たい (Want to do) Verb stem + たい. For wanting things, use 欲しい.

国くにに行いきたいです。

I want to go to the country.

新あたらしいコンピューターが欲ほしいです。

I want a new computer.

 〜なければならない (Must do) ない-form stem + なければならない. Short form: なきゃ.

行いかなければなりません。

I must go.

 あげる / くれる / もらう (Giving & Receiving) あげる = give (to others). くれる = give (to me). もらう = receive.

友達ともだちに本ほんをあげました。

I gave a book to my friend.

友達ともだちが本ほんをくれました。

My friend gave me a book.

友達ともだちから本ほんをもらいました。

I received a book from my friend.

 比べる (Comparison) Use もっと for "more". Use 同じ for "same".

これはもっと大おおきいです。

This is bigger.

これとそれは同おなじです。

This and that are the same.

 〜ましょう (Let's ~) Polite suggestion to do something together.

一緒いっしょに食たべましょう。

Let's eat together.

 から / ので (Because) から = because (casual/assertive). ので = because (softer).

仕事しごとがあるから、行いけません。

I can't go because I have work.

 もし〜なら (If) Conditional. もし adds emphasis.

もし時間じかんがあるなら、来きてください。

If you have time, please come.

第7: 聞く (Questions)

はい/いいえ Questions Add か to the end of a statement.

学生がくせいですか？ — はい、学生がくせいです。

Are you a student? — Yes, I am a student.

 何・誰・どこ・いつ・なぜ (WH-Questions) Place the question word where the answer would go.

何なにを食たべますか？

What will you eat?

誰だれが来きますか？

Who is coming?

学校がっこうはどこですか？

Where is the school?

第8: 時を言う (Expressing Time)

時の言葉 (Time Words) 今日, 昨日, 明日 don't need に. Other time words use に.

今日きょう、学校がっこうに行いきます。

I go to school today.

朝あさに食たべます。

I eat in the morning.

第9: よく使う文 (Common Patterns)

AはBです A is B.

私わたしは学生がくせいです。

I am a student.

 AはBが[形容詞]です A's B is [adjective].

この部屋へやは大おおきいです。

This room is big.

 AはBをVます A does V to B.

私わたしは本ほんを読よみます。

I read a book.

 場所にNがある/いる There is N at [place]. ある for things, いる for living things.

部屋へやに本ほんがあります。

There is a book in the room.

家いえに人ひとがいます。

There is a person in the house.

第10: 二つの話し方 (Formal vs Casual)

ます形 と 基本形 ます is polite (strangers, work). Basic form is casual (friends, family).

食たべます → 食たべる / 行いきます → 行いく / です → だ

Polite → Casual: eat / go / is
