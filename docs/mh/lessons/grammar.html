言こと葉ばの形かたち

言こと葉ばの来くる道みち
言こと葉ばの終おわりにする事ことが来くる。一ひとつにする言こと葉ばが何なにをするか見みせる。する事ことか、どんなか、物もの+だ で終おわる。

私わたしは 本ほんを 読よむ。

私わたしは 人ひとだ。

言いわない事こと
知しっている事ことは言いわない。誰だれが、何なにを - 知しるなら言いわなくていい。これがいつもの事こと。

「何なにを 飲のむ？」「水みず。」

ある / いる
ある事ことは持もつ事ことと一ひとつ。「持もつ」のする事ことはない。生いき物ものはいる、他ほかは全部ぜんぶある。

動物どうぶつがいる。

本ほんがある。

私わたしは 本ほんがある。

一ひとつにする言こと葉ば

は
何なにの事ことかを見みせる。「これの事ことは...」

この 本ほんは 良よい。

私わたしは 目めが 良よい。

が
する人ひと。誰だれがそれをするか。新あたらしい事こと。

誰だれが 来くる？

を
何なににする事ことをするか。

水みずを 飲のむ。

に
どこに行いく。いつ。どこにいる。

山やまに 行いく。

夜よるに 寝ねる。

家いえに 動物どうぶつがいる。

で
どこでする。何なにを使つかってする。に は物ものがいるところ、で はする事ことのところ。

家いえで 読よむ。

手てで 書かく。

の
物ものと物ものを一ひとつにする。する事ことを物ものにもする (五ごを見みる)。

私わたしの 本ほん。

山やまの 上うえ。

と
「と」で一ひとつにする。言いった事ことを書かく。言いう事こと、思おもう事こと、音おと、名前なまえ。

水みずと 本ほんを 買かった。

「行いく」と 言いった。

も
「も」で一ひとつ。は、が、を のの所ところに使つかう。

父ちちは 本ほんを 読よむ。母ははも 読よむ。

から / まで
始はじまるところ (から)、終おわるところ (まで)。から はなぜかも言いえる。

家いえから 山やままで 行いく。

夜よるまで 待まつ。

寒さむいから、家いえにいる。

より
Xより = 「Xよりもっと」。この一ひとつの言こと葉ばで全部ぜんぶ。

山やまより 大おおきい。

水みずより 食たべ物ものが 好すきだ。

する事ことの形かたち

三みっつと一ひとつの方かた
三みっつと一ひとつの方かたがある。る = 今いまと後あと。た = 前まえ。ない = しない事こと。なかった = 前まえにしなかった事こと。ます は人ひとへの言いい方かたで、言葉ことばの道みちではない。全部ぜんぶこの三みっつと一ひとつの上うえに作つくる。

食たべる → 食たべた → 食たべない → 食たべなかった

行いく → 行いった → 行いかない → 行いかなかった

する → した → しない → しなかった

来くる → 来きた → 来こない → 来こなかった

ての方かた
全部ぜんぶを一ひとつにする方かた。する事ことを一ひとつにする、欲ほしい事こと (+ください)、今いましている (+いる)、していい (+もいい)、してはいけない (+はいけない)。時ときは終おわりだけ。く→いて、む→んで、つ→って、る→て。

書かく → 書かいて

飲のむ → 飲のんで

待まつ → 待まって

食たべる → 食たべて

書かいて 読よんで 寝ねた。

待まってください。

今いま 読よんでいる。

座すわってもいい？

入はいってはいけない。

られる / える
できる事こと。る-のする事こと: る をなくして、られる を足たす。他ほか: え の方かた + る。する → できる。

書かく → 書かける

見みる → 見みられる

する → できる

どんな

い の言こと葉ば
する事ことと一ひとつ。寒さむい だけで言こと葉ばが終おわる。変かえる: くない、かった、くて。だ はいらない。

寒さむい。

寒さむくない。

寒さむかった。

寒さむくて 悲かなしい。

な の言こと葉ば
物ものの前まえに な を足たす。じゃない (違ちがう)、だった (前まえ)。

必要ひつような 物もの。

必要ひつようじゃない。

必要ひつようだった。

必要ひつようで 良よい。

〜くなる / 〜になる
変かわる事こと。全部ぜんぶの変かわり方かたがこれ一ひとつ。

暑あつい → 暑あつくなる。

必要ひつよう → 必要ひつようになる。

する事こと + 物もの
する事ことや言こと葉ばを物ものの前まえに入いれて、それを見みせる。言こと葉ば作づくりの強つよさ。

山やまで 動うごく 子供こども。

私わたしが 買かった 物もの。

物ものにする

の
いつもの方かた。食たべるのが好すき。聞きく時ときにも: 何なにしているの？

食たべるのが 好すきだ。

何なにしているの？

こと
する事ことを考かんがえにする。目めで見みえない事こと。「動うごく事こと」は考かんがえ。「飲のむ物もの」は手てで持もてる。この違ちがいが必要ひつよう。

動うごくことは良よい。

行いくことができる。

必要ひつようなことを言いう。

もの
手てで持もてるもの。本当ほんとうにある何なにか。「飲のむもの」= 飲のめる何なにか。「こと」は考かんがえ、「もの」は物もの。

飲のむもの。

読よむもの。

必要ひつようなものを持もっている。

方かた
どうするかの方かた。する事ことの始はじめに足たす。食たべ方かた= どう食たべるか。使つかい方かた= どう使つかうか。こと・もの・方かたの三みっつで全部ぜんぶ言いえる。

食たべ方かたを教おしえてください。

この言こと葉ばの使つかい方かたが知しりたい。

聞きき方かたが必要ひつようだ。

思おもうと「もし」

たい / 欲ほしい
する事ことがしたい (する事ことの始はじめ + たい)。物ものが欲ほしい (欲ほしい)。

行いきたい。

新あたらしい 家いえが 欲ほしい。

たら / なら / ければ
たら = もしそれが起おこったら (した + ら)。なら = もしそうなら。ければ = い の言こと葉ばの「もし」(い → ければ)。

雨あめが 来きたら、行いかない。

時ときがあるなら、来きてください。

寒さむければ、家いえにいる。

と思おもう / と言いう
と で何なんでも言いえる。思おもう事こと、言いう事こと、音おと。という = 「その名前なまえの」。

良よいと 思おもう。

「行いく」と 言いった。

大おおきい山やまという 所ところ

ために
何なにのためにするか。する事こと+ために か 物もの+のために。

知しるために 読よむ。

子供こどものために。

聞きき方かた

か / ね / よ
聞きく / そうでしょう？ / 強つよく言いう。いつもの事ことでは か を言いわない事ことも多おおい。

これは 水みずか？

良よい 日にちだね。

行いくよ。

何なに・誰だれ・どこ・いつ・なぜ + も/でも
聞きく言こと葉ばは言いう事ことのところに入いれる。+も (ない) = 何なにもない。+でも = 何なんでも。5つの言こと葉ば + 2つのつなぎ = 20の言こと葉ばが作つくれる。

何なにを 食たべる？→ 何なにも 食たべない。

何なんでも 作つくれる。

誰だれもいない。

誰だれでもできる。
