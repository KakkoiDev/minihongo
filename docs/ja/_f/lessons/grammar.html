言こと葉ばの形かたち (文の構造)

言こと葉ばの来くる道みち
述語が文末に来る。助詞が語の役割を示す。文は動詞・形容詞・名詞+だ で終わる。

私わたしは 本ほんを 読よむ。

私わたしは 人ひとだ。

言いわない事こと
既知の情報は省略する。主語・目的語など、文脈から分かるものは言わなくてよい。これが日本語の基本。

「何なにを 飲のむ？」「水みず。」

ある / いる
存在がそのまま所有を表す。「have」に当たる動詞は不要。生き物には「いる」、それ以外は「ある」を使う。

動物どうぶつがいる。

本ほんがある。

私わたしは 本ほんがある。

一ひとつにする言こと葉ば (助詞)

は (wa)
文の主題を示す。「〜については...」という意味。

この 本ほんは 良よい。

私わたしは 目めが 良よい。

が (ga)
動作の主体を示す。誰がそれをするか。新しい情報を表す。

誰だれが 来くる？

を (o)
動作の対象を示す。何に対して動作をするか。

水みずを 飲のむ。

に (ni)
方向・時間・存在の場所を表す。どこへ行くか、いつか、どこにいるか。

山やまに 行いく。

夜よるに 寝ねる。

家いえに 動物どうぶつがいる。

で (de)
動作の場所・手段を表す。「に」は存在の場所、「で」は動作の場所。

家いえで 読よむ。

手てで 書かく。

の (no)
名詞と名詞をつなぐ。動詞を名詞化する用法もある（第5章参照）。

私わたしの 本ほん。

山やまの 上うえ。

と (to)
並列・共同・引用を表す。発言・思考・音・名前など、あらゆるものを引用できる。

水みずと 本ほんを 買かった。

「行いく」と 言いった。

も (mo)
「〜も」で同様であることを示す。は・が・を の代わりに使う。

父ちちは 本ほんを 読よむ。母ははも 読よむ。

から / まで
起点（から）と終点（まで）を表す。「から」は理由も表せる。

家いえから 山やままで 行いく。

夜よるまで 待まつ。

寒さむいから、家いえにいる。

より
「Xより」で「Xよりも」の意味。この助詞一つであらゆる比較を表せる。

山やまより 大おおきい。

水みずより 食たべ物ものが 好すきだ。

する事ことの形かたち (動詞の活用)

三みっつと一ひとつの方かた
普通形のみ扱う。丁寧形（ます）は社会的な層であり文法ではない。る/た/ない/なかった。すべてこの上に構築する。

食たべる → 食たべた → 食たべない → 食たべなかった

行いく → 行いった → 行いかない → 行いかなかった

する → した → しない → しなかった

来くる → 来きた → 来こない → 来こなかった

ての方かた
万能の接続形。動作の連結、依頼（+ください）、進行（+いる）、許可（+もいい）、禁止（+はいけない）。時制は文末のみ。活用: く→いて、む→んで、つ→って、る→て。

書かく → 書かいて

飲のむ → 飲のんで

待まつ → 待まって

食たべる → 食たべて

書かいて 読よんで 寝ねた。

待まってください。

今いま 読よんでいる。

座すわってもいい？

入はいってはいけない。

られる / える
可能を表す。一段動詞: る を取って られる を付ける。五段動詞: え段 + る。する → できる。

書かく → 書かける

見みる → 見みられる

する → できる

どんな (修飾)

い の言こと葉ば
動詞と同様に活用する。「寒い」だけで文が成立する。活用: くない・かった・くて。「だ」は不要。

寒さむい。

寒さむくない。

寒さむかった。

寒さむくて 悲かなしい。

な の言こと葉ば
名詞の前に「な」を付ける。否定は「じゃない」、過去は「だった」。

必要ひつような 物もの。

必要ひつようじゃない。

必要ひつようだった。

必要ひつようで 良よい。

〜くなる / 〜になる
状態の変化を表す。あらゆる変化をこの一つの型で表現できる。

暑あつい → 暑あつくなる。

必要ひつよう → 必要ひつようになる。

する事こと + 物もの
動詞や文を名詞の前に置いて修飾する。語彙を作り出す力の源。

山やまで 動うごく 子供こども。

私わたしが 買かった 物もの。

物ものにする (名詞化)

の
くだけた名詞化。「食べるのが好き」。疑問文にも使う:「何しているの？」

食たべるのが 好すきだ。

何なにしているの？

こと
動作を抽象的な概念にする。「走ること」は走るという考え。「飲むもの」は物理的な物。「こと」は抽象、「もの」は具体。

動うごくことは良よい。

行いくことができる。

必要ひつようなことを言いう。

もの
触れたり指せたりする物理的なもの。「飲むもの」= 飲める何か。「こと」は抽象的な概念、「もの」は具体的な物。

飲のむもの。

読よむもの。

必要ひつようなものを持もっている。

方かた
やり方を表す。動詞の連用形に付ける。食べ方 = 食べる方法。使い方 = 使う方法。こと（概念）+ もの（物）+ 方（方法）で何でも表現できる。

食たべ方かたを教おしえてください。

この言こと葉ばの使つかい方かたが知しりたい。

聞きき方かたが必要ひつようだ。

思おもうと「もし」 (意志と条件)

たい / 欲ほしい
動作の希望（動詞の連用形 + たい）。物の希望（欲しい）。

行いきたい。

新あたらしい 家いえが 欲ほしい。

たら / なら / ければ
たら = もしその事態が起こったら（動詞のた形 + ら）。なら = もしそうであるなら。ければ = い形容詞の条件形（い → ければ）。

雨あめが 来きたら、行いかない。

時ときがあるなら、来きてください。

寒さむければ、家いえにいる。

と思おもう / と言いう
「と」で何でも引用できる。思考・発言・音。「という」は「〜と呼ばれる」の意味。

良よいと 思おもう。

「行いく」と 言いった。

大おおきい山やまという 所ところ

ために
目的と受益者を表す。動詞+ために、または名詞+のために。

知しるために 読よむ。

子供こどものために。

聞きき方かた (疑問と終助詞)

か / ね / よ
疑問（か）/ 同意の確認（ね）/ 主張（よ）。くだけた会話では「か」を省略して語尾を上げるだけのことも多い。

これは 水みずか？

良よい 日にちだね。

行いくよ。

何なに・誰だれ・どこ・いつ・なぜ + も/でも
疑問詞は答えの位置に置く。+も（否定）= 何もない・誰もいない。+でも = 何でも・誰でも。疑問詞5つ + 助詞2つ = 20語の代名詞が作れる。

何なにを 食たべる？→ 何なにも 食たべない。

何なんでも 作つくれる。

誰だれもいない。

誰だれでもできる。
